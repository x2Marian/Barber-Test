BookingWeb - Mini site pentru rezervari BarberSalon (GitHub Pages)

Cum publici pe GitHub Pages (cel mai simplu):
1. Creează un repository nou pe GitHub (public). Nume sugestiv: salon-booking
2. În repository, apasă "Add file" -> "Upload files" și urcă fișierele din acest folder (index.html, styles.css, script.js, README.md)
3. Mergi la Settings -> Pages -> Source -> Branch: main -> / (root) -> Save
4. După câteva secunde, site-ul va fi disponibil la:
   https://<username>.github.io/<repo>/

Notă: Acest mini-site nu salvează rezervările pe server (fără backend). Folosește WhatsApp pentru notificare.
